# 全局快捷键 (Front-end Spec C.1)

| 快捷键 | 操作 | 适用页面 |
|--------|------|----------|
| F8 | 图号联想 | 报价录入 |
| F2 | BOM 编辑 | BOM 树 |
| Ctrl+S | 保存草稿 | 报价/单据 |
| Ctrl+Enter | 提交审批 | 报价/审批 |
| Alt+T | 含税切换 | 金额字段 |
| N | 新建 | 工作台/列表 |
| R | 刷新 | 甘特/工作台 |
| Esc | 关闭抽屉 | 工作台 |
| Ctrl+K | 全局搜索 | 全局 |
| / | 全局搜索 | 全局 |
| Enter | 批量通过 | 报价审批 |

可在 **管理 → 快捷键设置** (`/admin/keyboard`) 自定义键位，保存至 `localStorage` 键 `erp-keyboard-shortcuts`。
