## 合同正文

### 第一条 合同依据

1.1 本合同依据《中华人民共和国民法典》《中华人民共和国著作权法》《中华人民共和国数据安全法》《中华人民共和国个人信息保护法》等相关法律法规签订。

1.2 甲方为 CNC 加工行业的离散制造企业，为提升数字化管理能力、解决"工序流转复杂、委外频繁、图纸核心、扫码追溯"等行业痛点，**委托乙方**为其定制开发昆山佰泰胜专属 ERP 系统。

1.3 乙方为具备行业专业软件交付能力的信息科技公司，拥有 PRD、架构设计、UI/UX 规格、移动端（Android 端）实施经验的工程化团队。

### 第二条 项目内容与范围

#### 2.1 项目目标

打造一套贴合 CNC 离散制造行业特点的 ERP 系统，**灵魂设计理念为"一码到底、一数到底"**——通过工单码（GD-）、流转码（LZ-）、设备码（SB-）三大码体系贯穿车间，让老师傅脑子里的"活地图"变成新人也能 5 分钟读懂的"明规则"。

#### 2.2 V1.0 交付范围

**13 个核心 Epic**（详见附件 A《昆山佰泰胜专属 ERP 系统产品需求文档（PRD）V1.3.7》）：

| Epic | 模块名称 | 优先级 |
|------|---------|--------|
| E1 | 基础设施与权限 | P0 |
| E2 | 客户与销售（报价/订单/合同回款）| P0 |
| E3 | 图纸与物料（BOM/工艺库）| P0 |
| E4 | 扫码仓储（APP 出入库）| P0 |
| E5 | 生产执行核心（工单+排产+扫码三码+委外下单）| P0 |
| E6 | 委外加工深化（7 状态机+返修闭环）| P0 |
| E7 | 品质管控（来料/FA 首件/三次元/不良品）| P0 |
| E8 | 采购管理 | P0 |
| E9 | 财务与成本（利润预警）| P0 |
| E10 | 人事管理 | P0 |
| E11 | 报表看板（生产工作台/委外面板/料号面板）| P0 |
| E12 | 委外到货与品质领料协同 | P0 |
| E13 | 供应商协同（163 邮箱 + 月度对账）| P0 |

**客户端**：
- **Web 端**：Vue 3 + Element Plus（PC 端 100 用户并发）
- **Android APP**：Kotlin + Jetpack（一线扫码端，含扫码三码 / 出入库扫码 / 报工 / 领料 / 仓管到货扫码）

**非功能需求**：

- 99.5% 可用性

- 字段级 AES-256 加密
- 操作审计日志

### 第六条 数据安全（数据安全报告）

> **【乙方郑重承诺】** 甲方图纸、工艺、BOM、客户/供应商档案、价格等所有数据，均属于甲方核心商业秘密。乙方以"本地化部署、最小授权、字段级加密、全链路审计"四大原则进行保护，确保**不泄露、不外传、不被乙方用于其他用途**。

#### 6.1 数据本地化部署

- **甲方自有服务器**：本系统部署在甲方指定的**本地物理服务器或甲方私有虚拟化环境**（如 VMware ESXi、Hyper-V），**不部署在任何公有云、乙方云端、第三方 SaaS 平台**。
- **服务器物理控制权归甲方**：机房进出、服务器开机/关机、网络接入、操作系统口令、数据库口令等均由甲方独立控制，乙方仅持有业务系统管理员账号。
- **甲方离线即可运行**：在甲方断网/断外网情况下，系统所有业务功能（扫码出入库、报工、查图纸、审批、报表等）均能正常运转；外网恢复后异步同步。
- **数据不出厂**：所有业务数据（图纸 PDF、BOM、报价单、工单、扫码记录、品质记录、财务数据）的存储、计算、备份**全部发生在甲方机房边界内**。

#### 6.2 字段级加密（图纸与敏感数据）

- **加密算法**：所有敏感字段采用 **AES-256-GCM**（或同等强度国密 SM4）对称加密，落盘前完成。
- **加密范围**：
  - 图纸文件（PDF / DWG / STEP）：文件级加密 + 存储路径混淆，仅授权角色可解密下载
  - 客户/供应商联系电话、价格、毛利率、合同金额：字段级加密存储
  - 数据库连接串、应用配置文件：密钥与口令分文件存储，运维脱敏
- **密钥管理**：数据库主密钥（DEK）由甲方 IT 管理员独立保管，乙方不持有甲方生产环境的明文密钥。

#### 6.3 权限与访问控制（最小授权）

- **角色化 RBAC**：内置 9 类系统角色（系统管理员/总经理/业务员/工程师/生管/仓管/品质/采购/财务/人事），按"岗位-角色-数据范围"三层授权。
- **数据范围隔离**：业务员只看自己跟进的客户；生管只看自己负责的车间；仓管只看到授权仓库的库存。
- **图纸访问留痕**：每次图纸下载、预览、转发均记录操作人 IP、MAC、时间、操作类型，**审计日志保留 5 年**。
- **离职即时回收**：甲方在员工离职流程触发后，IT 管理员可在 5 分钟内停用账号，所有"记住登录"自动失效。

#### 6.4 操作审计与防泄露

- **全量审计日志**：登录、查询、新增、修改、删除、导出、打印 7 类操作 100% 留痕，包含操作人、IP、时间戳、操作对象前后值。
- **水印与防截图**：图纸预览页与敏感页面叠加 **登录人姓名 + 工号 + 时间** 半透明水印，截图带水印可追溯。
- **导出审批**：图纸、BOM、客户清单等敏感数据**导出需主管审批**，审批通过后系统记录导出文件 SHA-256 与去向。
- **USB / 外发管控（可选）**：支持接入甲方 DLP/桌面管控，对图纸类文件外发进行阻断（需甲方提供客户端）。

#### 6.5 数据备份与容灾

- **本地三副本**：数据库每日全量备份 + 6 小时增量备份，备份文件同样加密存储。
- **异地冷备**：每周一次异地冷备（甲方指定第二地点存放加密硬盘），异地备份同样加密。
- **RPO ≤ 6 小时 / RTO ≤ 4 小时**：灾难场景下数据丢失不超过 6 小时，业务恢复不超过 4 小时。
- **乙方不持有甲方备份**：备份介质由甲方独立保管，乙方工程师协助恢复时**不接触明文备份**。

- **背景调查**：乙方参与本项目的工程师入职前均完成**无犯罪记录证明 + 竞业核查**。
- **保密协议**：所有参与人员签署**专项保密协议 + 离职后 3 年保密延续条款**，违约金不低于 ¥50,000 元（详见第 7.2 条）。

- **无凭据不外发**：乙方工程师远程支持时**仅通过甲方提供的堡垒机登录**，操作全程录屏录像，录像保存 1 年。
