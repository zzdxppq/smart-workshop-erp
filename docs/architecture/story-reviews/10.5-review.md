### 2.2 替换模式

```typescript
// Before
async function loadDetail(id: number): Promise<any> {
  const res = await fetch(`/api/v1/quotes/${id}`);
  return res.json();
}

// After
import type { QuoteDTO } from '@/api/generated/models/QuoteDTO';
async function loadDetail(id: number): Promise<QuoteDTO> {
  const res = await fetch(`/api/v1/quotes/${id}`);
  return res.json() as Promise<QuoteDTO>;
}
```

### 2.3 props / emits 类型补全

```typescript
// Before
defineProps<{ data: any }>();

// After
import type { OrderDTO } from '@/api/generated/models/OrderDTO';
defineProps<{ data: OrderDTO }>();
```

### 2.4 第三方库（ECharts）处理

- ECharts option 类型树深且与后端 DTO 不强对应
- 局部保留 `as any` 是业界通行做法
- AC-10.5.1 允许每文件 ≤ 1 个 `as any` 边界 ✅
