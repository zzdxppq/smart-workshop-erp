## 2. 技术评审

### 2.1 Gradle 版本选型

| 选项 | 评估 | 决策 |
|------|------|------|
| Gradle 8.7（AGP 8.5 兼容） | ✅ V1.3.7 1.4 build.gradle.kts 用 AGP 8.5 | ✅ 选 |
| Gradle 8.9 | 🟡 AGP 8.5 兼容但未实测 | ❌ 风险高 |
| Gradle 8.5 | 🟢 AGP 8.5 兼容但旧 | ❌ 失去 wrapper 升级意义 |

**决策**：Gradle 8.7 ✅

### 2.2 产物来源

- **gradlew / gradlew.bat**：Android Studio 自动生成（File → New → Project → Wrapper）
- **gradle-wrapper.jar**：与 Gradle 8.7 发行版绑定
- **gradle-wrapper.properties**：手写，distributionUrl 锁 `https://services.gradle.org/distributions/gradle-8.7-bin.zip`

### 2.3 SHA256 校验

- 官方 URL：`https://services.gradle.org/distributions/gradle-8.7-wrapper.jar.sha256`
- 校验流程：
  ```bash
  curl -fsSL https://services.gradle.org/distributions/gradle-8.7-wrapper.jar.sha256 | \
    tee android-impl/.gradle-wrapper-sha256.txt
  ```
- 落地后 CI 增加 step：`sha256sum -c android-impl/.gradle-wrapper-sha256.txt`

### 2.4 行尾与 .gitattributes

```gitattributes
# android-impl/.gitattributes
*.bat text eol=crlf
gradlew text eol=lf
```

→ 防止 Windows dev 提交 CRLF 污染 ✅
