## D.1 风险识别

| 风险类别 | 风险点 | 影响 |
|---------|--------|------|
| 数据外发 | 图纸 / BOM / 客户名单被下载后外发 | 客户流失、定向赔偿 |
| 内部越权 | 员工访问超出岗位的数据 | 商业秘密扩散 |
| 第三方泄露 | 乙方工程师将数据带走 | 知识产权流失 |
| 物理盗取 | 服务器 / 备份硬盘被盗 | 全部数据曝光 |
| 外部攻击 | SQL 注入 / XSS / 越权 | 系统被入侵 |

## D.2 防护体系（4 大原则 + 8 道防线）

### 防线 1：本地化部署

- 系统部署在**甲方自有物理服务器**（详见附件 C）
- **不部署在任何公有云、乙方云端、第三方 SaaS**
- 甲方断网时系统**离线可运行**
- 物理控制权（机房门禁、开机口令、数据库口令）**全部归甲方**

### 防线 2：字段级加密

- **算法**：AES-256-GCM（等同国密 SM4 强度）
- **加密范围**：
  - 图纸文件（PDF/DWG/STEP）→ 文件级加密 + 路径混淆
  - 客户/供应商电话、价格、毛利、合同金额 → 字段级加密
  - 数据库连接串、配置文件密钥 → 分文件存储
- **密钥管理**：数据库主密钥（DEK）由**甲方 IT 独立保管**，乙方不持有

### 防线 3：最小授权（RBAC）

- 9 类系统角色：系统管理员 / 总经理 / 业务员 / 工程师 / 生管 / 仓管 / 品质 / 采购 / 财务 / 人事
- 数据范围隔离：业务员只看自己客户、生管只看自己车间、仓管只看授权仓库
- **图纸访问留痕**：每次下载、预览、转发记录操作人 IP、MAC、时间

### 防线 4：审计与防泄露

- 7 类操作 100% 留痕（登录/查询/新增/修改/删除/导出/打印）
- **水印**：图纸预览页叠加"登录人姓名 + 工号 + 时间"半透明水印
- **导出审批**：图纸、BOM、客户清单导出需主管审批
- **USB / 外发管控**（可选）：接入甲方 DLP，图纸外发自动阻断

### 防线 5：备份与容灾

- **本地三副本**：每日全量备份 + 6 小时增量备份，均加密存储
- **异地冷备**：每周一次异地冷备（甲方第二地点）
- **RPO ≤ 6 小时 / RTO ≤ 4 小时**
- 备份介质由甲方独立保管，**乙方不接触明文备份**

### 防线 6：乙方人员约束

- 入职前完成**无犯罪记录 + 竞业核查**
- 签订**专项保密协议 + 离职后 3 年保密延续条款**（违约金 ≥ ¥50,000）
- 甲方有权对核心开发人员**背景审查与替换**
- 远程支持**仅通过甲方堡垒机登录**，全程录屏录像 1 年
- 乙方**零数据收集**：开发测试数据由乙方造数，**严禁使用甲方真实数据**

### 防线 7：网络隔离

- 服务器**禁止直连外网**
- 甲方内部网络通过 VLAN 隔离生产、办公、访客
- 远程运维走 VPN + 堡垒机 + 短信/动态口令双因子

### 防线 8：应急响应

- 数据泄露事件 1 小时内启动应急
- 乙方 4 小时内派员到场（昆山本地 ≤ 4 小时，国内 ≤ 24 小时）
- 协助甲方取证、定位、阻断、消除影响

## D.3 数据生命周期保护

| 阶段 | 保护措施 |
|------|---------|
| 采集 | HTTPS / 内网 TLS 1.3 传输，前端字段级加密 |
| 存储 | 数据库字段级 AES-256 加密 + 文件级加密 + 路径混淆 |
| 传输 | 内网专线 / VPN / 堡垒机跳板 |
| 使用 | RBAC 最小授权 + 水印 + 审计日志 |
| 共享 | 导出审批 + SHA-256 校验 + 去向登记 |
| 备份 | 加密 + 异地冷备 + RPO/RTO 监控 |
| 销毁 | 7 日内删除 + 销毁证明 + 介质消磁 |

## D.5 客户提问应答（FAQ）

**Q1：图纸会不会被乙方工程师带走？**
A：不会。乙方工程师在甲方堡垒机内操作，全程录屏录像 1 年；开发测试用数据由乙方造数，**严禁使用甲方真实数据**；所有参与人员签专项保密协议 + 离职后 3 年延续条款，违约金 ≥ ¥50,000 元。

**Q2：系统会不会被黑客攻击？**
A：本地化部署 + 不连外网 + 字段级加密 + RBAC + 审计 + WAF（可选）多层防御。

**Q3：服务器丢了怎么办？**
A：每日全量备份 + 异地冷备 + 全加密介质，**丢盘不解密**。RPO ≤ 6 小时，RTO ≤ 4 小时即可恢复业务。

**Q4：员工离职后能不能继续访问？**
A：5 分钟内停用账号即可。"记住登录"自动失效，全部会话立即吊销。

**Q5：乙方会不会偷偷用甲方数据做案例？**
A：详见合同第六条 6.7 款。乙方需在项目终止后 7 日内**删除所有甲方数据并提交《数据销毁证明》**，否则按泄漏论处，最低赔偿 ¥200,000 元。
