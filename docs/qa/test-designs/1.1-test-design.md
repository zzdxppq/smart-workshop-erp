```javascript
import http from 'k6/http';

export const options = { vus: 100, duration: '60s', thresholds: { http_req_duration: ['p(95)<200'], http_req_failed: ['rate<0.001'] } };
export default () => {
  const res = http.post('http://localhost:8080/api/v1/auth/login', JSON.stringify({username:'admin',password:'Admin@123'}), {headers:{'Content-Type':'application/json'}});
  check(res, { '200': r => r.status === 200 });
  sleep(1);
};
```

## 七、准入与退出条件

### 7.1 准入条件（Go / No-Go）

**Go 必须满足全部 7 条**：
1. Architect 评审 9.2/10 通过、3 条 P1 修补已在 dev commit 闭环
2. `application-platform.yml` 含 DEK 路径 / Redis 配置 / Flyway 迁移
3. Testcontainers MySQL 8 + Redis 7 启动成功
4. OpenAPI 契约 `auth/login` `users` `roles` schema 与实现 100% 对齐
5. 种子 9 类角色 + admin 账号幂等执行无报错
6. logback 配置禁止打印 `password_hash` / `phone` 明文
7. Postman 集合 10 用例全部可执行
