## 二、测例详细（14 测例）

### 业务域 A：认证 + 审批工作流（4 测例）

#### TC-10.2.A1 登录

- **文件**：`e2e/a/login.spec.ts`
- **链路**：浏览器打开 `/login` → 输入 `sales@btsheng.cn` + 密码 → 提交
- **端点覆盖**：`POST /auth/login`
- **断言**：
  ```typescript
  await expect(page).toHaveURL(/\/dashboard/);
  const token = await page.evaluate(() => localStorage.getItem('token'));
  expect(token).toBeTruthy();
  expect(token?.length).toBeGreaterThan(50);  // JWT
  ```
- **风险**：token 格式漂移 → 🟢 长度断言 + `/^eyJ/` 前缀
- **耗时**：< 3s

#### TC-10.2.A2 报价提交审批

- **文件**：`e2e/a/quote-submit.spec.ts`
- **链路**：登录 sales → 新建报价 → 填写 4 字段（客户/产品/数量/单价）→ 保存（DRAFT）→ 提交审批
- **端点覆盖**：`POST /quotes` + `POST /quotes/{id}/submit`
- **断言**：
  ```typescript
  await expect(page.locator('[data-testid="quote-status"]')).toHaveText('SUBMITTED');
  // 4 通道推送：邮件/站内信/企业微信/钉钉
  const logs = await getAuditLogs(quoteId);
  expect(logs).toContain('stream:notify triggered');
  ```
- **耗时**：< 4s

#### TC-10.2.A3 报价审批通过

- **文件**：`e2e/a/quote-approve.spec.ts`
- **链路**：登录 manager → 进待办 → 找到 A2 提交的报价 → 金额 < 20万（无二次密码）→ 通过
- **端点覆盖**：`GET /approvals/pending` + `POST /quotes/{id}/approve`
- **断言**：
  ```typescript
  await page.locator('[data-testid="approve-btn"]').click();
  await expect(page.locator('[data-testid="quote-status"]')).toHaveText('APPROVED');
  // 二次密码弹窗不出现（金额 < 20万）
  await expect(page.locator('[data-testid="high-amount-confirm"]')).not.toBeVisible();
  ```
- **耗时**：< 4s

#### TC-10.2.A4 报价转订单

- **文件**：`e2e/a/quote-to-order.spec.ts`
- **链路**：登录 sales → 进 A3 报价详情 → 点击"转化为订单"
- **端点覆盖**：`POST /quotes/{id}/convert-to-order`
- **断言**：
  ```typescript
  await page.locator('[data-testid="convert-btn"]').click();
  await expect(page).toHaveURL(/\/orders\/\d+/);
  const orderNo = await page.locator('[data-testid="order-no"]').textContent();
  expect(orderNo).toMatch(/^SO-\d{8}-\d{4}$/);
  ```
- **耗时**：< 3s

### 业务域 B：订单 + 利润（4 测例）

#### TC-10.2.B1 订单列表

- **文件**：`e2e/b/order-list.spec.ts`
- **链路**：登录 sales/manager/gm 三角色 → 进订单列表
- **端点覆盖**：`GET /orders`
- **断言**：
  ```typescript
  // sales 角色：仅 owner 单
  const salesRows = await page.locator('[data-testid="order-row"]').count();
  // manager 角色：dept 单
  // gm 角色：all 单
  // 数量递增：sales ≤ manager ≤ gm
  expect(salesRows).toBeLessThanOrEqual(managerRows);
  expect(managerRows).toBeLessThanOrEqual(gmRows);
  ```
- **耗时**：< 4s

#### TC-10.2.B2 订单审批

  await expect(page.locator('[data-testid="order-status"]')).toHaveText('PRODUCING');

#### TC-10.2.B3 转生产

#### TC-10.2.B4 利润分析

- **文件**：`e2e/b/order-profit.spec.ts`
- **链路**：登录 gm → 进 B2 订单详情 → 利润 tab
- **端点覆盖**：`GET /orders/{id}/profit`
- **断言**：
  ```typescript
  const revenue = await page.locator('[data-testid="revenue"]').textContent();
  const cost = await page.locator('[data-testid="cost"]').textContent();
  const margin = await page.locator('[data-testid="margin"]').textContent();
  expect(parseFloat(revenue!)).toBeGreaterThan(0);
  expect(parseFloat(cost!)).toBeGreaterThan(0);
  expect(parseFloat(margin!)).toBeCloseTo(parseFloat(revenue!) - parseFloat(cost!), 2);
  // 委外占比（与 1.45 报表一致）
  const outsourceRatio = await page.locator('[data-testid="outsource-ratio"]').textContent();
  expect(outsourceRatio).toMatch(/^\d+\.\d{2}%$/);
  ```
- **耗时**：< 5s

### 业务域 C：委外 + 返修（4 测例）

#### TC-10.2.C2 委外状态推进

## 三、CI 集成

### GitLab CI 配置

```yaml
# .gitlab-ci.yml (web-impl 仓新增)
e2e:
  stage: e2e
  image: mcr.microsoft.com/playwright:v1.49.0-jammy
  services:
    - name: erp-backend:latest
      alias: backend
  script:
    - npm ci
    - npx playwright install --with-deps chromium
    - npm run gen:api
    - npm run typecheck:ci
    - npx playwright test --project=chromium --reporter=junit,html
  artifacts:
    when: always
    paths:
      - playwright-report/
      - test-results/
    reports:
      junit: junit.xml
    expire_in: 30 days
  only:
    - merge_requests
    - main
  parallel: 4
```
