- **前置**：4 模板（GD-/LZ-/SB-/WW-）fixture 数据准备就位 · 2 OS（Windows + Linux）渲染完成
- **步骤**：
  1. 渲染 4 模板 × 2 OS = 8 张 PNG（baseline + 13.2 实装）
  2. `compare -metric AE baseline/label-gd-windows.png current/label-gd-windows.png diff.png`
  3. 重复 4 模板 × 2 OS = 8 次对比
- **期望**：
  - 全部 8 张图 diff < 0.1%（像素级一致）
- **断言**：
  ```bash
  for template in GD LZ SB WW; do
    for os in windows linux; do
      diff_count=$(compare -metric AE \
        baseline/label-${template,,}-${os}.png \
        current/label-${template,,}-${os}.png \
        diff.png 2>&1)
      # AE (Absolute Error) 像素差异数 · 400x300=120000 像素 · < 0.1% = < 120 像素
      test $diff_count -lt 120
    done
  done
  ```
