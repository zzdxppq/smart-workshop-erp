### 验证 5：行尾正确

```bash
cd android-impl
file gradlew
file gradlew.bat
test -f .gitattributes && cat .gitattributes
```

**期望**：
- `gradlew`: "Bourne-Again shell script, ASCII text executable"（LF）
- `gradlew.bat`: "DOS batch file, ASCII text, with CRLF line terminators"（CRLF）
- `.gitattributes` 包含：
  ```
  *.bat text eol=crlf
  gradlew text eol=lf
  ```

### 验证 6：distributionUrl 锁定

```bash

```

**期望**：
- `distributionUrl=https\://services.gradle.org/distributions/gradle-8.7-bin.zip`（官方）
- 或 `distributionUrl=https\://nexus.internal/gradle-8.7-bin.zip`（内网镜像，需 DevOps 确认）

### 验证 7：版本锁生效

```bash

sdk install gradle 8.10
sdk use gradle 8.10

cd android-impl
./gradlew --version  # 应显示 8.7
```

**期望**：本地 Gradle 8.10 不影响 `./gradlew` 行为 · 仍跑 8.7
