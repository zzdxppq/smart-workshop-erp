# 附录 A：金额阈值速查表

| 业务 | 阈值 | 审批人 | 备注 |
|------|------|--------|------|
| 报价 | < 5万 | 业务员自审 | - |
| 报价 | 5-20万 | 部门经理 | - |
| 报价 | > 20万 | 总经理 | 客户黑名单直接驳回 |
| 订单 | 同报价 | + 信用额度超限升级 | 信用额度校验叠加 |
| 采购 | < 1万 | 采购员自审 | - |
| 采购 | 1-5万 | 采购主管 | - |
| 采购 | > 5万 | 总经理 | - |
| 付款 | ≤ 10万 | 财务总监单签 | - |
| 付款 | > 10万 | 总经理 + 财务总监双签 | - |
