# 0. 灵魂一致性声明（PM 范蠡）

**为什么是 ERP？**：CNC 加工厂是典型的"看着粗、实际细"的离散制造业。车间里每一颗螺丝、每一刀切削、每一次换机台都直接影响交付与利润。通用 ERP 解决不了"图号怎么跟"、"工时怎么算"、"首件谁签"这些事。这套系统的存在，是把老师傅脑子里的隐性知识外化为系统规则，让新人和老人能在同一张图纸上协作。
