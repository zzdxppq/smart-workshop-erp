## 三、需客户确认的开放问题（共 2 项）

> **请木辛日匀于 2026-06-12 前书面回复（邮件 / 系统留言均可）**

### Q1 关于"77 状态"的真实含义

**我方理解**：贵方在评审会中提到的"77 个生命状态"，技术对齐后理解为：

- **7 个主状态**：草稿 / 待排产 / 已排产 / 加工中 / 已报工 / 已检验 / 已入库 / 已关闭（主流转 7 步）；
- **70 个状态相关字段**：每道工序状态（10 工序 × 4 字段：开工 / 完工 / 合格 / 报废）+ 时间戳 + 责任人 + 金额节点 ≈ 70 个字段；用于"一数到底"全维度追溯。

**实现策略**：状态机用 7 主状态枚举（DB enum + Spring State Machine），字段用 work_order_detail 表行级展开（每工序一行 × 多列字段），既好维护又能追溯到任意维度。

**请确认**：上述释义是否符合贵方业务理解？如贵方期望的是"77 个独立状态"，请提供完整状态清单，我方将重新评估实现工作量。

### Q2 公网 IP / 阿里云费用调研进展

**贵方待办**：@木辛日匀 调研运营商公网 IP / 阿里云 ECS 月费用。

**我方依赖**：调研结果决定 V1.1 阶段 2 部署启动时间（预计 2026-Q3 内）。

**请回复**：调研预计完成时间，以便我方排期对接 HTTPS 证书申请 + 反向代理配置 + 供应商 APP 远程访问联调。
