## C.1 应用服务器（必选 · 1 台）

| 配置项 | 推荐参数 | 用途 |
|--------|---------|------|
| CPU | Intel Xeon Gold 6326 / AMD EPYC 7402 以上，**8 核 16 线程** | 应用服务 + 报表计算 |
| 内存 | **32 GB DDR4 ECC** | 100 用户并发 + 报表缓存 |
| 系统盘 | 1 × 1 TB NVMe SSD（RAID 1） | 操作系统 + 应用 |
| 数据盘 | 2 × 2 TB SATA SSD（RAID 1） | 数据库 + 文件存储 |
| 网络 | 千兆网卡 × 2（双网卡绑定 / 内外网隔离） | 接入甲方内网 |
| 电源 | 冗余电源（1+1） | 7×24 稳定 |
| 形态 | 机架式 2U 或塔式 | 机房或独立机柜 |

## C.7 硬件采购与本合同的关系

1. **硬件不在合同范围**：硬件采购、运输、安装、调试费用由甲方独立承担，**不计入合同金额 ¥68,000 元**。
2. **乙方协助义务**：乙方提供：
   - C.1 ~ C.5 选型建议书
   - 部署前环境检查报告（机房 / 网络 / UPS / 备份）
   - 部署 + 调试服务（含在合同内）
3. **硬件质保**：由硬件原厂提供，**乙方不负责硬件维修**，但配合甲方 IT 在质保期内向原厂报修。
4. **数据安全要求**：所有承载本系统的服务器**禁止连接任何公有云、外网 SaaS 平台**；备份介质需使用**支持加密的型号**（硬件加密 + 软件加密双保险）。
