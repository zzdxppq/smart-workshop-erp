### §4.1 useOutsourceStateMachine.ts `as const` 字面量改造

**改动前**（V1.3.8 baseline · `export type` 字面量联合）：

```typescript
export type OutsourceState = 'DRAFT' | 'SUBMITTED' | 'ACCEPTED' | 'IN_PROGRESS' | 'COMPLETED' | 'REWORK' | 'CLOSED'
```

**改动后**（Sprint 13.5 IMPL · `as const` 字面量 + typeof 派生类型 · 与 codegen 联合类型结构同构）：

```typescript
export const OutsourceState = ['DRAFT', 'SUBMITTED', 'ACCEPTED', 'IN_PROGRESS', 'COMPLETED', 'REWORK', 'CLOSED'] as const
export type OutsourceState = (typeof OutsourceState)[number]
```

**优势**：
- 字面量数组单一来源（DRY · 修改一处即同步）
- `typeof stateSteps[number]` 派生类型与 codegen 联合类型 100% 结构同构
- 后续若增/减状态（V1.4）改一处即可
