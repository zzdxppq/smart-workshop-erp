## 3. Printers.vue 取消 any 兜底（证据）

### 3.1 修改前（baseline 临时 any）

```typescript
// Sprint 14 13.7 baseline-typecheck-fix 临时绕过：openapi.yaml 缺 12.2 Printer 端点定义
// 待 13.7 ship 后切回 codegen 强类型
// eslint-disable-next-line @typescript-eslint/no-explicit-any
const E12PrinterService: any = null
// eslint-disable-next-line @typescript-eslint/no-explicit-any
type SysPrinter = any
```

### 3.2 修改后（13.7 IMPL 强类型）

```typescript
import { E12PrinterService } from '@/api/generated/services/E12PrinterService'
import type { SysPrinter } from '@/api/generated/models/SysPrinter'
```

### 3.3 5 处方法内部强类型替换（grep 证据）

| 行 | 函数 | 改前 | 改后 |
|----|------|------|------|
| 262 | `reload()` | `const r: any = await E12PrinterService.listPrinters(...)` | `const r = await E12PrinterService.listPrinters(...)` |
| 287 | `onTest(row: SysPrinter)` | `const r: any = await E12PrinterService.testPrinterConnection(row.id)` | `const r = await E12PrinterService.testPrinterConnection(row.id)` |
| 350 | `onSubmit()` | `let r: any` | `let r: { code?: number; message?: string }` |
| 380 | `onDelete(row: SysPrinter)` | `const r: any = await E12PrinterService.deletePrinter(row.id)` | `const r = (await E12PrinterService.deletePrinter(row.id)) as { code?: number; message?: string }` |
| 397 | `toggleEnabled()` | `const r: any = await E12PrinterService.updatePrinter(...)` | `const r = (await E12PrinterService.updatePrinter(...)) as { code?: number; message?: string }` |

> 模板 `<template>` 中 `el-table-column #default="{ row }"` 因 Element Plus 内部 `DefaultRow` 类型不匹配，需在 4 处调用处加 `(row as SysPrinter)` 断言（typecheck:ci 验收通过）。
