## B.6 源代码与介质

- **交付时机**：验收通过后 7 个工作日内
- **交付物**：
  - 完整源代码（不含乙方通用框架库）Git 仓库 + 离线 ZIP 压缩包
  - 数据库 Schema + 初始数据 SQL 脚本
  - 部署脚本（Docker Compose 或裸机部署脚本）
  - 一键备份/恢复脚本
- **介质**：1 块 1TB 移动硬盘（加密）+ Git 仓库只读权限（甲方内部 GitLab）
- **知识产权**：详见主合同第六条
