### 2.3 灰度准入 / 暂停 / 回滚标准

| 状态 | 触发条件 | 处置 |
|------|---------|------|
| **准入（Go）**| ① 86 测例 100% PASS · ② typecheck:ci 0 错 · ③ 客户服务器就位 | 进入下一阶段 |
| **暂停（Hold）**| 任一阶段 P0 缺陷 ≥ 1 · 或核心业务（订单/PO/质检）阻塞 | 暂停灰度 · 修补后重测 |
| **回滚（Rollback）**| 同一 P0 缺陷在 24 小时内 2 次未修复 · 或业务冲击面 > 30% 角色 | feature flag 全置 false · 回退 V1.3.7 |
