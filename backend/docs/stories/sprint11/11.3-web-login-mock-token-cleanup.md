## 1. 现状（PRD 对齐检查）

```typescript
// src/views/auth/Login.vue 第 23 行
auth.setToken(r?.token || 'mock-token')
```

**问题**：
- mock-token 兜底让 dev 可不连后端测试 · 但 V1.3.8 部署后会导致"任意密码都能登录"
- 无 Loading 状态 · 提交期间无反馈
- 无错误提示 · 错误时控制台打印但 UI 不显示

## 2. 目标

| 改造 | 内容 |
|------|------|
| 1. 移除 mock 兜底 | `r?.token` 必须存在否则 throw new Error('登录失败：未返回 token') |
| 2. Loading 状态 | `loading = ref(false)` · 提交期间禁用按钮 + 显示 loading 图标 |
| 3. 错误提示 | Element Plus `ElMessage.error('账号或密码错误')` |
| 4. 401/403 区分 | 401 提示"账号或密码错误" · 403 提示"账号无权限" |
| 5. 字段校验 | 账号非空 + 密码 ≥ 6 位 + 前端 blur 校验 |

## 3. Acceptance Criteria

### AC-11.3.1 mock-token 兜底移除

测例：`grep -n "mock-token" src/views/auth/Login.vue` 退出码 1（无匹配）

### AC-11.3.2 Loading + 错误提示

测例：
- 提交期间按钮 disabled + loading icon
- 错误密码时 `ElMessage.error` 弹出
- 401 与 403 不同提示文案
