## 1. 现状（PRD 对齐检查）

```vue
<!-- src/views/auth/Login.vue 第 86-88 行 -->
<style scoped>
.login { max-width: 400px; margin: 100px auto; }
</style>
```

**问题**：
- 极简样式 · 无卡片 · 无居中垂直 · 无背景渐变 · 无品牌 LOGO · 无响应式
- 用户登录时看到的是纯白页面 + 简单输入框 · 不符合 UX Handoff §3.1"视觉门面"要求

## 2. 目标 UX

参考 UX Handoff `docs/ux-handoff.md` §3.1 登录页规范：

| 元素 | 要求 |
|------|------|
| 布局 | 垂直水平居中（flex 100vh） |
| 背景 | linear-gradient 135deg 蓝-白双渐变（V1.3.7 品牌色 #1890ff） |
| 卡片 | 圆角 12px 阴影 0 4px 24px rgba(0,0,0,0.08) · 白色背景 · padding 40px |
| 品牌 LOGO | 顶部 60×60 PNG + "昆山佰泰胜 ERP" 标题 |
| 输入框 | Element Plus `<el-input>` · 大号 · 圆角 |
| 按钮 | Element Plus `<el-button type="primary" size="large">` · 全宽 |
| 响应式 | < 768px 卡片宽 90vw · ≥ 768px 卡片宽 400px |
| 暗色模式 | (V1.3.9 backlog · 本 Story 不做) |

## 3. Acceptance Criteria

### AC-11.2.1 卡片 + 居中

测例：浏览器 1440×900 分辨率 · 卡片 400px 宽 · 垂直水平居中 · 圆角 12px · 阴影存在

### AC-11.2.2 渐变背景

测例：浏览器开发者工具查看 `<body>` 或 `.login-page` 容器 · `background: linear-gradient(135deg, #1890ff 0%, #ffffff 100%)`

### AC-11.2.3 品牌 LOGO

测例：登录卡片顶部 · 60×60 PNG + 文字"昆山佰泰胜 ERP" · 居中

### AC-11.2.4 响应式

测例：浏览器 DevTools 切到 iPhone 12 (390×844) · 卡片宽度 = 351px (90vw) · 全部元素可见无横向滚动
