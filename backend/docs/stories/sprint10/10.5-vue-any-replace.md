1. 5 个 .vue 中所有 `any` 替换为 `src/api/generated/models/*` 中的具体类型
2. 局部 `function (data: any)` 改为 `(data: QuoteDTO)` 等
3. props / emits 类型补全
4. `npm run typecheck`（vue-tsc）通过 · 0 错误

## 4. 风险

| 风险 | 等级 | 缓解 |
