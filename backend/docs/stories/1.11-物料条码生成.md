## Acceptance Criteria

### AC-4.1.1 — 物料条码生成（FR-4-1-1）
GIVEN 物料编码 WL-XXXX 已存在（来自 1.4/1.7/1.9）
WHEN 系统执行 `POST /api/v1/materials/{materialCode}/barcode`
THEN 返回 barcode_no=`BC{yyyyMMdd}{seq:4}`，payload 含 material_code + spec + 工艺 ID + 5 段成本

### AC-4.1.2 — 批量条码打印（FR-4-1-2）
GIVEN BOM 已就位（来自 1.9）
WHEN 工程师执行 `POST /api/v1/materials/batch-barcode` with `bomId`
THEN 系统展开 BOM 多级树生成 1-100 条 barcode（按 target_qty 缩放）

### AC-4.1.3 — 扫码解析（FR-4-1-3）
GIVEN APP 扫码设备扫描到 barcode
WHEN APP 调用 `GET /api/v1/materials/barcode/{barcodeNo}`
THEN 返回物料详情 + 工艺路线 + 5 段成本 + 库存提示（预留 1.12 hook）
