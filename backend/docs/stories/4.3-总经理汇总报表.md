## 2. Acceptance Criteria

### AC-4.3.1 汇总聚合

**Given** 总经理调用 `GET /api/v1/reports/gm-summary?period=LAST_30D`
**When** 系统聚合 30 天数据
**Then** 返回：
```json
{
  "code": 0,
  "data": {
    "period": "LAST_30D",
    "no_order_po_count": 12,
    "no_order_po_amount": 186500.00,
    "urgent_replenish_count": 5,
    "amount_threshold_passed_rate": 0.87,
    "procurement_manager_workload": 23,
    "outsource_cost_ratio": 0.18,
    "trend_chart": [
      {"date": "2026-05-15", "no_order_count": 0, "amount": 0},
      {"date": "2026-06-13", "no_order_count": 1, "amount": 12000}
    ]
  }
}
```
**And** HTTP 200，权限校验：仅 `GM` + `PROCUREMENT_MANAGER` 角色可见
**And** period 支持：`LAST_7D` / `LAST_30D` / `LAST_90D` / `CUSTOM`

### AC-4.3.2 缓存性能

**Given** 报表端点配置 Redis 5min 缓存（key=`gm:summary:{period}`）
**When** 5min 内第二次访问
**Then** cache HIT，P95 < 50ms
**And** 缓存失效：4.1 PO 创建 / 4.2 审批完成触发 `@CacheEvict`（AFTER_COMMIT）
**And** K6 压测 50 并发 P95 < 500ms（缓存未命中场景）

## 4. 技术实现要点

### 4.1 聚合 SQL（草图 · dev 细化）

```sql
SELECT
  COUNT(CASE WHEN source_type='NO_ORDER' THEN 1 END) AS no_order_count,
  SUM(CASE WHEN source_type='NO_ORDER' THEN amount ELSE 0 END) AS no_order_amount,
  COUNT(CASE WHEN purchase_reason='URGENT_REPLENISH' THEN 1 END) AS urgent_count,
  -- 审批通过率
  SUM(CASE WHEN status='APPROVED' AND amount>10000 THEN 1 ELSE 0 END) /
  NULLIF(SUM(CASE WHEN amount>10000 THEN 1 ELSE 0 END), 0) AS pass_rate,
  -- PROCUREMENT_MANAGER 工作量
  (SELECT COUNT(*) FROM sys_workflow_event
   WHERE approver_role='PROCUREMENT_MANAGER'
     AND created_at >= :start_date) AS pm_workload
FROM crm_purchase_order
WHERE created_at >= :start_date AND created_at < :end_date;
```

### 4.2 Redis 缓存

```java
@Cacheable(value = "gm:summary", key = "#period", ttl = 300)  // 5min
public GmSummaryDTO getSummary(String period) { ... }

@CacheEvict(value = "gm:summary", allEntries = true)
public void onPurchaseChange() { ... }  // AFTER_COMMIT
```

### 4.3 权限

```java
@PreAuthorize("hasAnyRole('GM', 'PROCUREMENT_MANAGER')")
@GetMapping("/api/v1/reports/gm-summary")
public Result<GmSummaryDTO> gmSummary(...) { ... }
```
