## 3. 资源嵌入方案

### 3.1 字体选择

| 字体 | 许可证 | 文件大小 | 推荐 |
|------|--------|---------|------|
| **Noto Sans CJK SC Regular** | Apache 2.0（SIL Open Font License）| ~16MB（Variable Font）| ✅ 选用 · 跨平台保真度最高 |
| Source Han Sans SC（思源黑体）| Apache 2.0 | ~16MB | 与 Noto 同源 · Adobe 版 |
| 文泉驿微米黑 | GPL | ~5MB | ❌ 许可证不兼容商业 |

**选用 Noto Sans CJK SC Regular.otf**（Google 版 · 与 Source Han Sans 同源不同名）。

### 3.2 资源路径

```
backend/src/main/resources/fonts/NotoSansCJK-Regular.otf   (16MB · jar 内)
backend/src/main/resources/fonts/NotoSansCJK-Bold.otf      (16MB · 可选)
```

### 3.3 服务端统一入口

```java
// backend/src/main/java/.../core/render/FontProvider.java (新增)
@Component
public class FontProvider {
    private final Font regularFont;
    private final Font boldFont;
    
    @PostConstruct
    public void init() throws IOException, FontFormatException {
        // 单一权威源 · jar 资源加载 · 不依赖系统字体路径
        try (InputStream isRegular = getClass().getResourceAsStream("/fonts/NotoSansCJK-Regular.otf")) {
            this.regularFont = Font.createFont(Font.TRUETYPE_FONT, isRegular);
            this.regularFont = regularFont.deriveFont(Font.PLAIN, 14f);
        }
        try (InputStream isBold = getClass().getResourceAsStream("/fonts/NotoSansCJK-Bold.otf")) {
            this.boldFont = Font.createFont(Font.TRUETYPE_FONT, isBold);
            this.boldFont = boldFont.deriveFont(Font.BOLD, 14f);
        }
    }
    
    public Font getRegular(float size) {
        return regularFont.deriveFont(size);
    }
    
    public Font getBold(float size) {
        return boldFont.deriveFont(size);
    }
}
```
