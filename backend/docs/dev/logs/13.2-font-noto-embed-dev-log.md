## 2. FontProvider API 设计

### 2.1 类签名

```java
@Component
public class FontProvider {
    public static final String FONT_RESOURCE_PATH = "/fonts/SourceHanSansCN-Normal.ttf";

    @PostConstruct
    public synchronized void init();     // 同步加载 · ~200ms · 不阻塞 Spring 启动（10MB TTF 解析）

    public Font getRegular(float size);  // AWT PNG 渲染 · Regular 派生
    public Font getBold(float size);     // AWT PNG 渲染 · Bold 派生
    public BaseFont getPdfBaseFont();    // iText/OpenPDF PDF 渲染 · IDENTITY_H + EMBEDDED
    public boolean isLoaded();           // 监控/测试断言
    public int getFontSizeBytes();       // 测例断言 [8MB, 12MB]
}
```

### 2.2 加载逻辑

```java
@PostConstruct
public synchronized void init() {
    try {
        loadFont();
    } catch (Exception e) {
        log.error("[FontProvider] 思源黑体初始化失败 · 降级到 JDK SansSerif fallback · error={}", ...);
        fallback();
    }
}

private void loadFont() throws IOException, FontFormatException {
    try (InputStream is = getClass().getResourceAsStream(FONT_RESOURCE_PATH)) {
        if (is == null) {
            throw new IOException("字体资源未找到: " + FONT_RESOURCE_PATH + " · 请确认 ... 存在");
        }
        // 1) 字节缓存（用于 PDF BaseFont.createFont 字节数组重载）
        ByteArrayOutputStream baos = new ByteArrayOutputStream();
        byte[] buf = new byte[8192];
        int n;
        while ((n = is.read(buf)) > 0) baos.write(buf, 0, n);
        this.fontBytes = baos.toByteArray();

        // 2) AWT PNG 渲染字体
        try (InputStream fontStream = getClass().getResourceAsStream(FONT_RESOURCE_PATH)) {
            this.regularFont = Font.createFont(Font.TRUETYPE_FONT, fontStream)
                    .deriveFont(Font.PLAIN, DEFAULT_FALLBACK_SIZE);
        }
        this.boldFont = regularFont != null
                ? regularFont.deriveFont(Font.BOLD, DEFAULT_FALLBACK_SIZE)
                : null;

        // 3) iText/OpenPDF BaseFont（IDENTITY_H + EMBEDDED）
        this.pdfBaseFont = BaseFont.createFont(
                FONT_RESOURCE_PATH, BaseFont.IDENTITY_H, BaseFont.EMBEDDED,
                false, fontBytes, null);
    }
    this.loaded = true;
}
```

### 2.3 Fallback 模式（部署前字体未就位时）

| API | 字体加载成功 | 字体加载失败（fallback） |
|-----|-------------|------------------------|
| `getRegular(size)` | 思源黑体 Regular(size) | `SansSerif PLAIN(size)` |
| `getBold(size)` | 思源黑体 Regular derive BOLD(size) | `SansSerif BOLD(size)` |
| `getPdfBaseFont()` | BaseFont(IDENTITY_H, EMBEDDED) | `null`（PdfA4Generator 内部 fallback Helvetica） |
| `isLoaded()` | `true` | `false` |
| `getFontSizeBytes()` | ~10MB | `0` |

> **关键设计**：fallback 不阻塞渲染 · 启动期间 logger 告警 · PM 可监控 `isLoaded()` 状态。

## 4. 12.4 PdfA4Generator 改造

### 4.1 改造点

```java
// BEFORE（12.4）
bodyCell.addElement(new Paragraph("[" + ... + "]",
        FontFactory.getFont(FontFactory.HELVETICA, 7)));  // 内置 Helvetica · 无中文
bodyCell.addElement(new Phrase(safe(line),
        FontFactory.getFont(FontFactory.HELVETICA, 6)));   // 中文显示为方框

// AFTER（13.2）
private Font cjkFont(float size, int style) {
    if (fontProvider != null && fontProvider.getPdfBaseFont() != null) {
        return new Font(fontProvider.getPdfBaseFont(), size, style);  // 思源黑体 IDENTITY_H
    }
    return new Font(FontFactory.HELVETICA, size, style);  // fallback Helvetica
}
bodyCell.addElement(new Paragraph(... , cjkFont(7f, Font.NORMAL)));
bodyCell.addElement(new Phrase(... , cjkFont(6f, Font.NORMAL)));
```

### 4.2 PDF 字体嵌入（AC-13.2.3）

`BaseFont.createFont(FONT_RESOURCE_PATH, BaseFont.IDENTITY_H, BaseFont.EMBEDDED, false, fontBytes, null)`

- `IDENTITY_H`：Unicode H 编码 · 支持中文（CJK Unified Ideographs U+4E00-U+9FFF）
- `EMBEDDED`：嵌入字体到 PDF 文件 · 接收端（Windows/macOS/Linux/Android PDF 阅读器）无需本地字体

预期 PDF 体积增量 ~200KB（嵌入思源黑体子集）。
