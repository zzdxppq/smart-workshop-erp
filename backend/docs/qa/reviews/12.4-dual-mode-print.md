## 1. 测试设计原则

| 维度 | 策略 | 理由 |
|------|------|------|
| **单元** | ProtocolAdapter 3 型号 + Socket + PDF 排版 | 8 测例（ZPL/TSPL 字节流断言 + 27 标签 PDFBox 解析）|
| **集成** | 7 端点 + ZPL/TSPL mock 打印机 | 10 测例（端点 + sys_print_log 落库 + 补打链）|
| **E2E** | web-impl + android-impl `<PrintButton>` | 6 测例（1/多/未 三态弹窗 + 三仓 UI 一致）|
| **风险** | Socket 代理 + 三仓 UI 一致性 | **risk-profile**（comprehensive 必做）|
| **性能** | 异步 + PDF 生成 < 500ms | 4 测例（Socket 3s / PDF 500ms / 线程池 4-16）|
| **安全** | 补打链防递归 + 错误码 50203 | 4 测例（referenceLogId / 多租户 / 错误码不暴露内网）|

## 3. risk-profile（comprehensive 必做）

| 风险 | 等级 | 触发场景 | 缓解 | 测例 | 负责人 |
|------|------|----------|------|------|--------|
| **Socket 代理卡死主线程** | 🔴 P0 | 客户机房 IP 不可达 · Socket `connect()` 阻塞数十秒 | `@Async` 独立线程池 + 3s 硬性超时 + `try-with-resources` | TC-12.4.1.5 / 1.8 | dev |
| **三仓 UI 不一致** | 🟡 P1 | web-impl 弹`<el-dialog>` · android-impl 弹`<AlertDialog>` · 模式选择不同 | web/android 共享 `label_template` preview 字段 + 文案字典 + Playwright + Espresso 截图对比 | TC-12.4.5.1-6 | web-impl / android-impl |
| **协议适配错误（型号 1 错 2 对）** | 🔴 P0 | Zebra ZD420 写 ZPL 字节流错（如 `^BC` vs `^B3`）| 3 型号字节流断言单测 + PDFBox 解析 QR 验证 | TC-12.4.1.1-3 | dev |
| **补打链无限递归** | 🟡 P1 | 原始 SUCCESS → 补打 b（ref=a）→ 补打 c（ref=b）→ ... | `referenceLogId != null` 拒绝 40954 · `@Transactional` 内校验 | TC-12.4.3.6 | dev |
| **PDF 排版错误（27 vs 30）** | 🟡 P1 | dev 沿用 Story §3 写 3×10=30 · 实际 3×9=27 | PDFBox 解析 `PdfPTable.size()==27` 验证 · 9 行/页实测 | TC-12.4.2.1-2 | dev |
| **异步失败补偿** | 🔴 P0 | @Async 线程抛 IOException · 异常被吞 · 审计漏失 | `@Async` 异常统一写 `sys_print_log.status=FAILED` + `error_msg=e.getMessage()` · HTTP 200 + body 含 `printLogId` | TC-12.4.1.7 | dev |
| **协议字段不匹配** | 🟢 低 | sys_printer.protocol 写错（如 "zpl" 小写）| ProtocolFactory 注入 + 字典 PRINTER_PROTOCOL 强校验 | TC-12.4.1.4 | dev |

**风险处置优先级**：
- 🔴 P0：Socket 卡死 · 协议错配 · 异步失败补偿（3 项）· 阻塞上线
- 🟡 P1：三仓 UI · 补打链 · PDF 排版 · IP 变化（4 项）· 文档化决策 · 不阻塞
- 🟢 低：协议字段（1 项）· 字典约束即可

## 8. 安全 / 合规

- **补打链防递归（不可循环打印）**：`referenceLogId != null` → 拒绝 40954 · 防死链

- **PDF 流 Content-Disposition 头**：`attachment; filename="labels-{log_no}.pdf"` · 防 XSS 文件名注入
