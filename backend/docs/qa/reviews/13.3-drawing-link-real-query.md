## 1. 测试设计原则

| 维度 | 策略 | 理由 |
|------|------|------|
| **5 类 link 真实查询** | 5 类各 1 正例 + 3 反例 | JOIN 路径 + 权限过滤 + 索引命中 |
| **Redis 缓存** | 端点 2/3 命中 + TTL=300s + @CacheEvict + 降级 | 5min TTL 一致性 + 3 次失败退化 DB |
| **性能** | V58 部分索引 + 1000 并发 P99 + 缓存命中 < 5ms + DB QPS | EXPLAIN 验证 + JMeter 压测 |
| **灰度** | 5 类 flag 协同 + 跨角色隔离 + admin 全开 | 复用 12.1 `draw.acl.gray.{ROLE}` |
| **E2E** | web-impl + android-impl 真实数据 | 替代 mock · 端到端 |
