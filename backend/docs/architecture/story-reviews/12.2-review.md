### 2.2 心跳探活

```java
// PrinterHeartbeatTask.java · @Scheduled 单线程 + TCP Socket 探活
@Component
public class PrinterHeartbeatTask {

    @Scheduled(fixedRate = 60_000)  // 每 60s 执行
    public void probe() {
        // 1. 查询所有 enabled=1 + type='LABEL' 的打印机（单线程串行）
        List<SysPrinter> printers = printerMapper.findByEnabledAndType(true, "LABEL");

        // 2. 逐台 TCP Socket 探活（2s 超时 · 串行执行）
        for (SysPrinter p : printers) {
            try (Socket s = new Socket()) {
                s.connect(new InetSocketAddress(p.getIp(), p.getPort()), 2000);
                p.setStatus("ONLINE");
                p.setFailCount(0);
                p.setLastHeartbeatAt(LocalDateTime.now());
            } catch (IOException e) {
                p.setFailCount(p.getFailCount() + 1);
                if (p.getFailCount() >= 2) {  // 容差 · 连续 2 次失败才标 OFFLINE
                    p.setStatus("OFFLINE");
                }
            }
            printerMapper.updateHeartbeat(p);
        }
        // 3. NORMAL 类型保持 status=UNKNOWN（OS 打印队列无 IP · 不探活）
    }
}
```

**关键点**：
- `@Scheduled(fixedRate = 60_000)` · 60s 周期
- `Socket.connect(InetSocketAddress, 2000)` · 2s connect 超时（避长卡）
- `fail_count >= 2` 容差 · 单次失败不立即标 OFFLINE · 防瞬断误标
- 单线程串行 · V1.3.9 客户规模 5-10 台 · V1.4 可改线程池
- NORMAL 类型跳过探活 · status 保持 UNKNOWN

### 2.4 打印时打印机选择逻辑

```typescript
// web-impl: PrintButton.vue (12.4 共用)
async function onPrintClick(codeType: 'GD' | 'LZ' | 'SB' | 'WW') {
  // 1. 调 /printers/available?type=LABEL
  const res = await axios.get('/api/v1/printers/available', { params: { type: 'LABEL' } });

  // 2. 三态判断
  if (res.data.count === 0) {
    ElMessage.warning('管理员先配置标签打印机');
    return;
  }
  if (res.data.count === 1) {
    return doDirectPrint(res.data.printers[0]);  // 1 台直打
  }
  // 3. 多台弹选择框
  printerSelectorRef.value.show(res.data.printers);
}
```

**后端逻辑**（`/available?type=LABEL`）：
- 0 台 → `count=0` · 前端提示"管理员先配置"
- 1 台 → `printers=[P1]` · 前端直接打印
- N 台 → `printers=[P1, P2, ...]` · 前端弹 `<el-dialog>` 选
